## ✏️ 如何添加/更新位置

### 方式一：GitHub Web 界面编辑（推荐）

1. 访问：https://github.com/Stoic42/superbrain-board
2. 进入 `data/locations.json`
3. 点击 ✏️ Edit 按钮
4. 添加或修改位置信息：

```json
{
  "id": "member-id",
  "name": "成员姓名",
  "role": "角色",
  "location": "城市/地区",
  "coordinates": [纬度, 经度],
  "description": "描述信息",
  "status": "active",
  "icon": "👨‍💻"
}
```

5. 提交更改
6. 等待 1-2 分钟，刷新页面查看更新

### 方式二：查找坐标

可以使用以下工具查找准确的经纬度：
- [OpenStreetMap](https://www.openstreetmap.org/) - 点击位置，查看 URL 中的坐标
- [百度地图坐标拾取](https://api.map.baidu.com/lbsapi/getpoint/index.html)
- [高德地图坐标拾取](https://lbs.amap.com/tools/picker)

**坐标格式**：`[纬度, 经度]`（注意顺序）

### 示例：添加新成员位置

```json
{
  "id": "newmember",
  "name": "新成员",
  "role": "学员 - 软件组",
  "location": "北京",
  "coordinates": [39.9, 116.4],
  "description": "软件开发，前端专家",
  "status": "active",
  "icon": "👨‍🎓"
}
```

## 💡 使用场景

1. **了解团队分布**：快速查看团队成员所在城市
2. **规划线下活动**：根据地理位置安排聚会或工作坊
3. **硬件协作**：知道谁在附近，方便硬件调试和组装
4. **时区协调**：了解时区差异，合理安排会议时间
   - 中国成员：UTC+8，适合中国时间上午9点-晚上9点
   - 美国东部（乐老师）：UTC-5/-4，适合中国时间晚上8点-早上8点
   - 美国西部（散修）：UTC-8/-7，适合中国时间晚上11点-早上11点
   - **最佳会议时间**：中国时间晚上8-9点（美国东部早上8-9点，西部早上5-6点）
