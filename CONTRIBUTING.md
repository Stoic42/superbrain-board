# 贡献指南

## 编辑方式

### 方式一：GitHub Web 界面（推荐）

1. 找到要编辑的文件
2. 点击 ✏️ Edit 按钮
3. 修改内容
4. 提交更改

### 方式二：本地编辑

1. Fork 本仓库
2. Clone 到本地
3. 创建分支：`git checkout -b feature/your-feature`
4. 提交更改：`git commit -am 'Add feature'`
5. 推送分支：`git push origin feature/your-feature`
6. 提交 Pull Request

## 数据格式

- JSON 文件必须符合 JSON 格式
- 提交前会自动验证
- 如有错误，GitHub Actions 会提示

## 博客格式

使用 Markdown 格式，支持：
- 标题、列表、链接
- 代码块
- 图片（使用相对路径或完整 URL）

## 问题反馈

如有问题，请提交 Issue。
