# 超脑共创看板

这是超脑项目的共创看板，通过 GitHub Pages 自动部署。

## 🚀 快速开始

访问：https://stoic42.github.io/superbrain-board

## ✏️ 如何编辑

### 更新项目数据

1. 进入 `data/projects.json`
2. 点击 ✏️ Edit 按钮
3. 修改内容并提交

### 更新看板任务

1. 进入 `data/board.json`
2. 添加/修改卡片
3. 提交后自动显示

### 发布博客

1. 在 `content/posts/` 创建新的 Markdown 文件
2. 文件名格式：`YYYY-MM-DD-title.md`
3. 更新 `content/posts/index.json`
4. 提交后自动显示

### 上传文件

1. 进入 `uploads/images/`
2. 点击 "Upload files"
3. 上传后获取 URL
