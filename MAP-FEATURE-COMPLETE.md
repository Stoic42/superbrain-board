## ✏️ 如何添加新位置

1. 访问：https://github.com/Stoic42/superbrain-board
2. 编辑 `data/locations.json`
3. 添加新的位置条目：

```json
{
  "id": "member-id",
  "name": "成员姓名",
  "role": "角色",
  "location": "城市/地区",
  "coordinates": [纬度, 经度],
  "description": "描述信息",
  "status": "active",
  "icon": "👨‍💻"
}
```

4. 提交更改
5. 等待 1-2 分钟，刷新页面查看
